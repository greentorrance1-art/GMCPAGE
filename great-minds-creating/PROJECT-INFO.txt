SETUP REQUIRED
================================================================

Before deployment, you MUST:

1. Create Firebase project
2. Configure Firebase credentials in config/firebase-config.js
3. Set up Stripe account
4. Add Stripe publishable key to js/merch.js
5. Add your logo images to assets/logos/
6. Update content (songs, videos, social links)

Complete instructions in SETUP-GUIDE.md
